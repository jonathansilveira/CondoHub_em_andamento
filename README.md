# SmartCondo 🏢

> Plataforma inteligente para gestão condominial, controle de acesso e comunicação entre moradores.

## 📖 Sobre o Projeto

O **SmartCondo** é uma plataforma completa de gestão condominial desenvolvida para modernizar e centralizar todas as operações de condomínios residenciais e comerciais.

A solução integra controle de acesso facial e veicular, monitoramento em tempo real, comunicação interna, gerenciamento de correspondências, ocorrências, aplicativo para moradores e diversas outras funcionalidades em uma única plataforma.

O sistema é composto por um **painel administrativo web** e um **aplicativo mobile**, oferecendo praticidade, segurança e uma melhor experiência para administradores, porteiros e moradores.

## 🚀 Funcionalidades

### 🔐 Controle de Acesso Inteligente

* Reconhecimento facial.
* Controle de acesso veicular.
* Cadastro e gerenciamento de credenciais.
* Carteirinhas digitais.
* Histórico completo de entradas e saídas.
* Dashboard com monitoramento em tempo real.
* Controle de acessos por níveis de permissão.
* Registro e auditoria de eventos.

---

### 👤 Níveis de Acesso

O sistema permite definir diferentes níveis de acesso, garantindo maior segurança e flexibilidade.

Perfis previstos:

* Básico
* Médio
* Completo
* Prestador de Serviço
* Van Escolar
* Visitante
* Funcionário

Cada perfil poderá possuir permissões específicas de acesso, horários permitidos e restrições configuráveis.

### 📦 Gestão Inteligente de Correspondências

* Registro de encomendas e correspondências.
* Controle de encomendas aguardando retirada.
* Notificação automática ao morador via aplicativo.
* Geração de QR Code para retirada rápida.
* Retirada simplificada, sem necessidade de cadastro de saída.
* Após a retirada, a correspondência é removida da lista ativa e armazenada automaticamente em um histórico/backup.
* Consulta ao histórico de correspondências retiradas.
* Otimização de desempenho através da separação entre registros ativos e históricos.
